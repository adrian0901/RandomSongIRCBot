# RandomSongIRCBot
a random song IRC bot.

# Settings
$chan - channel to connect to.

$server - server to connect to.

$port - port to connect to.

$nick - nick to connect with.

$ident - ident to connect with.

$gecos - meant for real name, can be used for anything

$music - music list
